# Simple Interest Calculator

## Project Title
**Simple Interest Calculator**

## Description
The Simple Interest Calculator is a beginner-friendly project designed to compute simple interest based on the principal amount, rate of interest, and time period. This project helps users quickly determine the total interest earned or payable using the standard simple interest formula.

**Formula Used:**

```
Simple Interest = (Principal × Rate × Time) / 100
```

This project is useful for students learning basic finance concepts and for anyone needing a quick calculation tool.

## Installation Instructions
Follow these steps to set up the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Yug-D-Shah/github-final-project.git
   ```

2. Navigate to the project directory:
   ```bash
   cd simple-interest-calculator
   ```

3. Run the program:
   - If using **Python**:
     ```bash
     python simple_interest.py
     ```
   - If using **JavaScript (Node.js)**:
     ```bash
     node simpleInterest.js
     ```

> Make sure the required runtime (Python or Node.js) is installed on your system.

## Usage
1. Enter the **principal amount**.
2. Enter the **rate of interest** (in percentage).
3. Enter the **time period** (in years).
4. The program will display the calculated **simple interest** and total amount.

This tool can be used for:
- Educational purposes
- Basic financial planning
- Quick manual calculations replacement
